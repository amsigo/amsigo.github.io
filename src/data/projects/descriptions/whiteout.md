# 프로젝트 소개
이 프로젝트는 화이트 아웃 서바이벌의 가짜 광고 버전을 모작하여   
실제 게임으로 제작한 프로젝트입니다.

이 게임은 전투원을 소환하여 곰을 사냥하고 고기를 획득하여 고기를 요리 합니다.   
획득한 고기는 스테이크로 구워 손님들에게 제공하고 보상 골드를 획득합니다.

프로젝트의 리소스는 에셋을 구매하여 조합하여 사용하였고   
애니메이션은 Mixamo에서 무료 애니메이션을 사용하였습니다.

## 몬스터 AI 구현
몬스터와 전투원의 AI는 Unity6의 Unity Behaviour를 이용하여   
BT로 행동 패턴을 설계하여 구현 하였습니다.
