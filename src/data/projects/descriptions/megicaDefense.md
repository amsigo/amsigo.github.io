# 프로젝트 소개

이 프로젝트는「닌자 대전 : 디펜스 RPG」를 레퍼런스로 하여   
마법사 콘셉트의 디펜스 RPG로 기획·개발한 프로젝트입니다.

메인 영웅과 동료 영웅으로 부대를 편성하고 마법을 사용하여   
몰려 오는 적 몬스터를 처치하여 스테이지를 클리어 하는 디펜스 게임입니다.

본 프로젝트는 프로토타이핑을 목적으로 빠르게 개발된 프로젝트로   
전체 콘텐츠 완성보다는 핵심 게임 시스템과 구조 검증에 중점을 두고 구현되었습니다.   
이로 인해 일부 콘텐츠 및 디테일한 요소는 미완성 상태로 남아 있습니다.

게임에 사용된 리소스는 상용 에셋을 활용하거나,
UI 레이아웃을 직접 설계한 뒤   
UI 에셋을 적용하였으며, 일부 리소스는 직접 디자인하여 제작하였습니다.

게임 데이터는 Excel 테이블 기반으로 구성되었으며,
이를 JSON 파일로 변환 후 게임 내에서   
파싱하여 해당 데이터를 기준으로 게임이 동작하도록 시스템을 설계 및 구현하였습니다.
