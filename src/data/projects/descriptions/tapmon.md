# 프로젝트 소개

텝몬은 터치로 몬스터를 처치하여 영웅을 성장시키는 클리거 게임입니다. 

몬스터를 처치하여 챕터의 스테이지를 모두 클리어하면 보스가 등장하며   
보스를 처치하면 다음 챕터로 넘어가며 테마가 변경됩니다.

콘텐츠로는 영웅 강화, 몬스터 뽑기, 몬스터 합성, 몬스터 도감 등의 콘텐츠가 있습니다.  

해당 프로젝트는 고등학생 시절 여름방학 팀 프로젝트 게임으로 개발 하였으며   
학교장배 게임 제작 대회에서 은상을 수상하였습니다.

### 영웅 강화
영웅을 강화 시켜 영웅을 성장시킵니다.


### 몬스터 뽑기
뽑기를 통해 획득한 몬스터는 전투에 참가시켜 버프를 얻거나 스킬 공격을 할 수 있습니다.

### 몬스터 합성
2마리의 몬스터를 합성하여 더 높은 랭크의 몬스터를 획득 할 수 있습니다.

### 몬스터 도감
획득한 몬스터는 도감에 기록됩니다.   

## 담당 역할
- 전투 시스템 구현
- 챕터 시스템 구현
- 버프 시스템 구현

- 게임 테이블 데이터 xml기반 데이터 관리 기능 구현
- 영웅 강화 구현
- 몬스터 합성 구현
