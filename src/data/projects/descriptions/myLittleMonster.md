# 개발 환경
- Unity
- NGUI
- Spine

# 프로젝트 소개
마이 리틀 몬스터는 몬스터로 팀을 구성하여 몬스터의 기본 공격 및 스킬과   
유저의 터치, 긋기, 흔들기를 이용하여 몬스터를 무찌르는 게임입니다.   

탐험에서 몬스터를 물리치고 수집한 알을 부화시켜 몬스터를 획득합니다.   
160종 이상 다양한 몬스터들이 존재하며 수집한 몬스터들로 팀편성을 할 수 있습니다.   

각 몬스터들은 스킬이 존재하며 몬스터들을 강화, 진화하여 성장 시킬 수 있습니다.

# 담당 역할
팀원 구성은 기획 1명, 디자인 1명 프로그래밍 1명으로 전체 프로그래밍을 담당 하였습니다

# 주요 구현 내용

### UI Frame Work
UIPopup과 UILayout으로 구분 지어 아웃게임 메뉴에서 나오는   
Layout 타이틀, 인벤토리, 몬스터 정보창 등을 UIPopup Class를 상속 받도록 하고   
UILayout을 각 Popup들을 조합하여 만들 수 잇는 구조로 구현하였습니다.

뒤로 가기 등을 고려하여 UILayout의 호출 스택을 쌓아   
이전 UILayout으로 돌아 갈 수 있는 형태로 구현하였습니다.

### 스킬 시스템 구현
스킬의 효과와, 상태이상의 효과를 각각 구현하여 기획자분이 Excel에 각 효과들을 조합하여   
 새로운 스킬의 데이터를 정의 할 수 있게 제작하여 Excel파일을 xml로 변환하여 사용 하였고   
 스킬이 데이터 기반으로 동작 할 수 있게 제작 하였습니다.
