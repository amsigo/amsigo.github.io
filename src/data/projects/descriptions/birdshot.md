# 개발 환경
- Unity

# 프로젝트 소개
버드샷은 스와이프 벽돌깨기와 같은 공대신 새를 날려서 벽돌을 깨는 벽돌깨기 게임 입니다.   
새는 색상이 있으며 같은 색상인 벽돌은 통과되고 다른 색상은 충돌하여 벽돌의 체력을 깍습니다.   

각 스테이지에는 미션이 있으며 미션을 클리어하면 스테이지가 클리어 됩니다.   
스테이지를 클리어하기 위해 전략적으로 새의 순서를 정해야 합니다.

## 캐릭터 스텟
아래와 같은 캐릭터 스탯을 업그레이드 할 수 있습니다.   
- 잠재력 : 새의 최대 슬롯 수 증가
- 각도 : 던질 수 있는 각의 크기 증가
- 기회 : 던질 수 있는 카운트 수 증가
- 경험치 : 새의 최대 슬롯 수 증가


## 사용 기술 / SDK
- Facebook SDK 연동
- AWS 랭킹 서버 구현
- 에셋번들 패치 시스템 구현

## 담당 역할
기획 1명, 프로그래밍 1명, 디자인 1명으로 팀이 구성되었고 프로그래밍을 담당하였습니다.   

## 구현 내용
### 인게임
- 벽돌 깨기 로직 구현
- 스테이지 동작 (스테이지 로드, 미션)
- 스테이지 제작 에디터 구현 (Unity Custom Editor로 구현)
- 스테이지 몯, 무한 모드 구현

### 아웃게임
- 게임 플레이를 위한 하트 시스템 구현
- 캐릭터 스텟 업그레이드 기능 구현
- 알 부화 시스템 구현
- 출석 보상 시스템 구현
